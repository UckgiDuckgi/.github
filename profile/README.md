## 🔥 주요 기능

1. 최저가 상품 __추천__ 및 __검색__
2. __통합 장바구니__ 결제
3. 소비내역 기반 __혜택__ 추천
4. __과거 소비__ 를 분석하여 __소비 목표__ 설정 및 관리
